    Omnideck v1.0 Quick Reference

    One-page cheat sheet for the all-in-one trading system

    üéØ The 10 Systems

    # System What It Does Display
    1 TD Sequential Exhaustion counter (9-count) Green/red numbers 1-9
    2 Squeeze Detector Volatility breakout signals Arrows on chart
    3 Liquidity Sweeps Stop hunt detection LL/HH tags
    4 EMA Trio Moving averages (50/100/200) Three lines on chart
    5 SuperTrend Adaptive trend ribbon Green/red band
    6 BMSB Bull Market Support Band 20 SMA + 21 EMA
    7 Regime Box Market mood classifier Background color
    8 Supply/Demand Zones Institutional price zones Horizontal boxes
    9 Candlestick Patterns 16 classic patterns Labels on chart
    10 Meta Tools System integration layer Behind the scenes

    üìä System Layers

    Layer Systems Purpose
    Layer 1: Trend EMA Trio, SuperTrend, BMSB Direction analysis
    Layer 2: Patterns TD Sequential, Squeeze, Candlesticks Signal detection
    Layer 3: Zones Regime Box, Supply/Demand, Liquidity Sweeps Key areas
    Layer 4: Integration Meta Tools Coordination

    ‚ö° Quick Interpretation Guide

    Getting Started (First 2 Weeks)

    • Enable ONLY 3 systems: EMA Trio + Regime Box + TD Sequential
    • Watch for alignment: All 3 agree = high-confidence setup
    • Disable the rest: Add more systems gradually as learning progresses

    Reading the 4 Layers

    • Layer 1 (Trend): Which direction? Use EMA Trio/SuperTrend/BMSB
    • Layer 2 (Patterns): Signal present? Look for TD/Squeeze/Candlesticks
    • Layer 3 (Zones): At key level? Check Regime/Supply-Demand/Liquidity
    • Layer 4 (Meta): All systems aligned? Meta Tools show confluence

    System Combination Examples

    • Trend + Pattern: SuperTrend green + TD Sequential 9 = Continuation setup
    • Pattern + Zone: Squeeze breakout + Supply Zone = High-probability entry
    • All 3 Layers: EMA crossover + TD 9 + Regime shift = Maximum confluence

    ü§ù Best System Combinations (Within Omnideck)

    Beginner Stack (3-4 Systems)

    • EMA Trio + Regime Box + TD Sequential: Trend + mood + exhaustion
    • SuperTrend + Squeeze + Supply/Demand: Trend + breakout + zones
    • TD Sequential + Candlestick + BMSB: Exhaustion + patterns + support

    Intermediate Stack (5-6 Systems)

    • Trend Set: EMA Trio + SuperTrend + BMSB
    • Signal Set: TD Sequential + Squeeze + Candlesticks
    • Zone Set: Regime Box + Supply/Demand + Liquidity Sweeps

    Advanced Stack (7+ Systems)

    • All 10 enabled: Use Meta Tools to filter confluence
    • Focus on layer alignment: Trend + Pattern + Zone all agree
    • Toggle on/off as needed: Disable visual clutter, keep key systems

    With Other Signal Pilot Indicators

    • Omnideck + Pentarch: Omnideck trend + Pentarch cycle phase
    • Omnideck + Janus Atlas: Omnideck signals + Janus price levels
    • Omnideck + Volume Oracle: Omnideck patterns + Volume confirmation

    ‚öôÔ∏è Recommended Settings

    Beginner Configuration ‚≠ê

    • Enable: EMA Trio, Regime Box, TD Sequential, Supply/Demand (4 systems)
    • Disable: All others
    • Why: Clean chart, easy to understand, covers trend/mood/exhaustion/zones
    • Best for: First 2-4 weeks, learning curve gentle

    Day Trader Configuration

    • Enable: SuperTrend, Squeeze, TD Sequential, Liquidity Sweeps (4 systems)
    • Disable: Long-term systems (BMSB, weekly-focused)
    • Why: Fast signals, breakout detection, exhaustion, stop hunts
    • Best for: 5m-1H charts, intraday trading

    Swing Trader Configuration

    • Enable: EMA Trio, BMSB, Regime Box, Supply/Demand, TD Sequential (5 systems)
    • Disable: Fast signals (Squeeze, Liquidity Sweeps)
    • Why: Trend confirmation, zone analysis, cycle exhaustion
    • Best for: 4H-Daily charts, multi-day holds

    Advanced Configuration (All 10 Systems)

    • Enable: All 10 systems
    • Adjust opacity: Lower visual clutter with transparency settings
    • Use Meta Tools: Let Meta layer filter for confluence
    • Best for: Experienced traders, larger screens, complex analysis

    ‚ö†Ô∏è When NOT to Use

    ‚ùå As first indicator ever (too complex for absolute beginners)
    ‚ùå Enabling all 10 systems at once (information overload)
    ‚ùå On slow devices with all systems enabled (performance issues)
    ‚ùå Without understanding individual systems first (learn components before combining)

    üîß Troubleshooting

    "Chart looks too cluttered"
    ‚Üí Disable half the systems - start with 3-5 systems max
    ‚Üí Use opacity settings to reduce visual weight
    ‚Üí Focus on one layer at a time (Trend OR Pattern OR Zone)

    "Systems giving conflicting signals"
    ‚Üí Normal - not all systems agree at once
    ‚Üí Wait for 2-3 systems to align before acting
    ‚Üí Meta Tools layer helps filter for confluence

    "Indicator running slowly"
    ‚Üí Disable unused systems (only enable what's needed)
    ‚Üí Reduce bars loaded in TradingView settings
    ‚Üí Use higher timeframes (Daily faster than 1m)

    "Not sure which systems to use"
    ‚Üí Start with beginner stack (EMA + Regime + TD + Supply/Demand)
    ‚Üí Add one system per week as learning progresses
    ‚Üí Match systems to trading style (see Settings section)

    Weekly ‚úÖ Good EMA Trio, BMSB, TD Sequential (long-term focus)

    üéØ Omnideck Success Tips

    1. Start with 3-5 systems max ‚Äî Add more gradually, avoid overload
    2. Wait for layer alignment ‚Äî Trend + Pattern + Zone all agree = best setups
    3. Disable visual noise ‚Äî Only enable systems actively using
    4. Learn individual systems first ‚Äî Understand components before combining
    5. Use system stacks ‚Äî Beginner/Day Trader/Swing Trader configurations (see Settings)

    üìñ Key Concepts

    10 Systems: TD Sequential, Squeeze, Liquidity Sweeps, EMA Trio, SuperTrend, BMSB, Regime Box, Supply/Demand, Candlesticks, Meta Tools
    4 Layers: Trend Analysis (Layer 1), Pattern Detection (Layer 2), Zone Analysis (Layer 3), Integration (Layer 4)
