    Augury Grid v1.0 ‚Äî Multi-Symbol Scanner¶

    Real-time scanner displaying MACD histogram crossovers with confluence filters across multiple assets simultaneously. Monitor up to 40 tickers from single dashboard view.

    üëã

    New to Trading? Start Here

    In Plain English: Instead of checking charts one-by-one to find signals, Augury Grid shows you 8-40 markets at once in a clean table format. Think of it like:

    • A radar screen showing all nearby aircraft
    • A security monitor showing all camera feeds
    • A sports scoreboard showing all games

    What You'll See:

    A grid/table showing multiple symbols with their current status:

    • Symbol Name (e.g., BTC, ETH, AAPL, SPY)
    • Signal Direction (‚Üë Bullish or ‚Üì Bearish)
    • Confluence Score (how many filters align)
    • Take Profit Level (TP1) and current P&L
    • Gauss Ribbon Status (trend filter)

    What Signals Does It Show?

    Augury Grid detects MACD histogram crossovers filtered by multiple confluence factors:

    • ‚Üë Bullish Signal = MACD histogram crosses above zero with confluence confirmation
    • ‚Üì Bearish Signal = MACD histogram crosses below zero with confluence confirmation
    • Confluence Filters: ADX (trend strength), Higher Timeframe alignment, OBV (volume), RSI (momentum), Distance from moving average
    • Score Display = Shows how many confluence factors align (higher = stronger signal)
    • ‚ö™ NO SIGNAL = MACD histogram hasn't crossed or confluence filters not met

    Perfect For:

    • Scanning multiple crypto pairs quickly
    • Finding the strongest setups across stocks
    • Monitoring your watchlist without clicking through charts
    • Spotting which markets are showing signals RIGHT NOW
    üéì What's MACD histogram?

    MACD (Moving Average Convergence Divergence) is a momentum indicator that shows the relationship between two moving averages. The histogram shows the difference between the MACD line and its signal line:

    • Histogram crosses above zero = Bullish momentum developing
    • Histogram crosses below zero = Bearish momentum developing

    Augury Grid doesn't just show raw MACD crosses‚Äîit adds confluence filters (ADX, HTF, OBV, RSI, distance) to show only the highest-probability setups.

    You don't need to understand the technical details! Augury Grid analyzes all the confluence factors and shows you a simple score and direction for each symbol.

    Beginner Starting Point:

    1. Start with 4-8 symbols you're familiar with (don't overwhelm yourself with 40)
    2. Watch for ‚Üë Bullish or ‚Üì Bearish signals with high confluence scores
    3. Click any symbol in the grid to jump directly to that chart
    4. Use it as a filter‚Äîonly look at charts showing active MACD crossover signals

    üí° Pro Tip: Augury Grid is perfect for crypto traders monitoring 10+ pairs, or stock traders scanning their watchlist for MACD momentum shifts. It saves you from clicking through dozens of charts looking for setups.

    üéØ Scanning Strategies¶

    Strategy 1: Broad Market Scan¶

    Approach:
    - Monitor 20-40 symbols across multiple sectors
    - Scan for any bullish (‚Üë) or bearish (‚Üì) MACD signals with high confluence scores
    - Focus attention on symbols showing 4/5 or 5/5 confluence

    Use Case: Opportunity discovery across diverse markets

    Example Watchlist: Major indices, sector ETFs, top crypto, forex pairs, commodities

    Strategy 2: Sector Rotation Scan¶

    Approach:
    - Monitor 10-15 symbols within specific sector
    - Compare relative confluence scores and momentum direction
    - Identify sector-wide momentum patterns

    Use Case: Sector-specific analysis and comparison

    Example Watchlist: All semiconductor stocks (NVDA, AMD, INTC, TSM, AVGO, QCOM, MU, etc.)

    Strategy 3: Correlation Scan¶

    Approach:
    - Monitor related assets (e.g., BTC + major altcoins)
    - Observe if MACD signals appear simultaneously or in sequence
    - Identify correlation patterns in momentum shifts

    Use Case: Understanding market leadership and correlation

    Example Watchlist: BTC/USD + top 15 altcoins

    Strategy 4: Portfolio Monitor¶

    Approach:
    - Track symbols in current portfolio holdings
    - Monitor for MACD momentum shifts on held positions
    - Use confluence scores as position management reference

    Use Case: Portfolio-level awareness of momentum changes

    Example Watchlist: Personal holdings (stocks, crypto, etc.)

    üìö Interpreting Grid Patterns¶

    Pattern 1: Broad Market Momentum¶

    Observation:
    Multiple symbols (5+) show bullish or bearish MACD signals with high confluence (4-5/5) simultaneously across same sector or asset class.

    Interpretation:
    Sector-wide or market-wide momentum shift in progress.

    Example: 8 out of 10 tech stocks showing bearish signals with 4-5/5 scores ‚Üí Broad tech sector momentum reversal characteristics.

    Pattern 2: Divergent Signals¶

    Observation:
    Some symbols show bullish signals while related symbols show bearish signals.

    Interpretation:
    Divergence in momentum direction within sector or asset class.

    Example: BTC showing bullish 5/5 signal, ETH showing bearish 4/5 signal ‚Üí Divergent momentum between major crypto assets.

    Pattern 3: Sequential Cascade¶

    Observation:
    MACD signals appear across related symbols in sequence over time rather than simultaneously.

    Interpretation:
    Potential leadership or lagging pattern within sector.

    Example: Tech mega-caps signal first (AAPL, MSFT), then mid-caps signal days later ‚Üí Potential cascade pattern in momentum shifts.

    Pattern 4: Isolated High-Probability Signal¶

    Observation:
    One symbol shows signal with 5/5 confluence while all others remain neutral or show low-confluence signals.

    Interpretation:
    Asset-specific momentum shift with exceptional confluence rather than broad market pattern.

    Example: Single stock shows bullish 5/5 signal while entire sector neutral ‚Üí Isolated high-probability setup characteristics.

    ‚ùì Frequently Asked Questions¶

    Q: How many symbols should I monitor?¶

    A: Depends on monitoring purpose. Focused scanning (10-15 symbols) allows detailed attention. Broad scanning (30-40 symbols) enables opportunity discovery. Individual preferences vary.

    Q: Can I use different timeframes for different symbols?¶

    A: No. All symbols in grid use the timeframe of the active chart. To scan multiple timeframes, open multiple chart windows with Augury Grid on each.

    Q: Do symbols need to be from same market?¶

    A: No. Can mix any asset types available on TradingView (stocks, crypto, forex, futures, indices, commodities, etc.)

    Q: What happens if a symbol has no data?¶

    A: Cell displays "No Data" message. Symbol may be unavailable on current TradingView plan or exchange.

    Q: Can I save multiple watchlists?¶

    A: Yes. Save different indicator configurations with different symbol lists. Switch between saved templates as needed.

    Q: Does grid slow down performance?¶

    A: Minimal impact. Optimized to efficiently monitor up to 40 symbols without performance degradation.

    Q: Can other users see my watchlist?¶

    A: No. Symbol lists stored in personal indicator settings, not visible to others.

    Q: How is this different from TradingView's native screener?¶

    A: Augury Grid displays on chart with continuous real-time updates, while native screener requires manual refresh and opens separately. Grid format provides at-a-glance monitoring of MACD confluence methodology with live performance metrics (TP1, P&L).

    Q: Can I export grid data?¶

    A: Grid data can be captured via TradingView alert webhooks. Each alert includes symbol and signal type data.

    Q: Does it work on mobile?¶

    A: Yes. Grid layout adapts to screen size. Mobile displays fewer symbols per row for readability.
