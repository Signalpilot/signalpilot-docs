    Changelog (High Level)¶

    This page tracks major version releases and updates across the Signal Pilot Suite. For detailed technical changes and migration steps, see the individual indicator documentation pages.

    2025‚ÄëQ1 - Signal Pilot Suite¶

    The Signal Pilot Suite represents a complete rebrand and modernization of the Signal Pilot indicator collection. This release brings significant improvements in performance, visual clarity, and educational compliance.

    What's New¶

    • Complete Suite Rebrand: All indicators modernized with consistent naming, visual design, and educational framing
    • Enhanced Performance: Optimized calculations across all indicators for faster load times and reduced resource usage
    • Improved Visual Clarity: Decluttered interfaces, better color contrast, and streamlined on-chart displays
    • Compliance Updates: All documentation and signal descriptions now use "potential" qualifiers for educational compliance
    • Expanded Documentation: Comprehensive guides, comparison tables, and step-by-step tutorials added

    Core Indicators¶

    Pentarch v1.0 ‚Äî Reversal Events

    What Changed:

    • 5-phase potential reversal system: TD (Touch Down) ‚Üí IGN (Ignition) ‚Üí WRN (Warning) ‚Üí CAP (Capitulation) ‚Üí BDN (Breakdown)
    • Enhanced 5-minute market profile detection for improved signal quality
    • New Volume Assist integration for confirmation of potential reversals
    • Cleaner UI with reduced visual clutter on chart
    • Boolean outputs added for easier alert configuration

    Key Features: Potential reversal detection, multi-phase signal system, volume confirmation, customizable alerts

    Janus Atlas v1.0 ‚Äî Levels System

    What Changed:

    • Expanded to 39 level types (up from 28 in previous versions)
    • New FVG/OB v2 detection with improved accuracy
    • Added breaker blocks, session levels, and VWAP variants
    • Decluttered visuals with optional level hiding
    • Boolean outputs for all level types
    • Comprehensive alert system for level interactions

    Key Features: Fair value gaps, order blocks, breakers, session levels, institutional levels, VWAP analysis

    Omnideck v1.0 ‚Äî Everything Indicator

    What Changed:

    • All-in-one system combining 10 indicators into a unified dashboard
    • New regime detection with anchored boxes showing market conditions
    • Squeeze/Expansion pulse indicators for potential volatility shifts
    • Unified settings panel for easier configuration
    • Improved performance with consolidated calculations
    • Modular design allowing individual components to be enabled/disabled

    Key Features: Composite momentum, trend analysis, volume flow, regime detection, multi-indicator dashboard

    Augury Grid v1.0 ‚Äî Multi-Symbol Screener

    What Changed:

    • 8-symbol real-time screener with quality scoring system
    • New potential TP1 (Take Profit 1) projection calculations
    • P&L tracking for educational performance monitoring
    • Multi-timeframe scanning (M1, M5, M15, H1 support)
    • Color-coded quality indicators for quick assessment
    • Customizable symbol watchlists

    Key Features: Multi-symbol monitoring, quality scores, potential target projections, real-time alerts

    Volume & Flow Analysis¶

    Volume Oracle v1.0 ‚Äî Volume Strategy

    What Changed:

    • Potential volume spike detection using Z-score statistical filtering
    • New Higher Timeframe (HTF) gating for context confirmation
    • Session validation to filter signals by trading hours
    • Complete position management system with entry/exit logic
    • Visual markers for volume anomalies
    • Customizable sensitivity thresholds

    Key Features: Volume spike detection, statistical filtering, session awareness, position management

    Plutus Flow v1.0 ‚Äî Advanced OBV

    What Changed:

    • Advanced On-Balance Volume (OBV) with CVD (Cumulative Volume Delta) divergence detection
    • New EVR (Extreme Volume Rejection) indicator for potential exhaustion signals
    • FlipGuard integration for potential breakout confirmation
    • Enhanced divergence detection with customizable lookback periods
    • Visual flow direction indicators
    • Improved alert system for divergence events

    Key Features: OBV analysis, divergence detection, volume exhaustion signals, flow confirmation

    Oscillators¶

    Harmonic Oscillator v1.0 ‚Äî Composite Momentum

    What Changed:

    • 3-in-1 composite oscillator combining MACD, RSI, and Stochastic RSI
    • New voting system that considers agreement across all three components
    • Adaptive thresholds that adjust to market conditions
    • FlipGuard integration for potential trend change confirmation
    • Unified momentum display with consensus indicators
    • Overbought/oversold detection with multi-indicator validation

    Key Features: Composite momentum, voting system, adaptive thresholds, multi-indicator confirmation

    Breaking Changes¶

    • Signal Renaming: The "OUT" signal has been renamed to "BDN" (Breakdown) across all indicators for clearer educational framing. Existing alerts using "OUT" will need to be updated.
    • Alert Message Format: Alert messages now include "potential" qualifiers (e.g., "Potential reversal detected" instead of "Reversal detected"). Update any automation scripts that parse alert text.
    • Boolean Output Names: Some boolean output variable names have changed for consistency. Check custom scripts that reference indicator outputs.
    • Default Settings: Several indicators have updated default settings for improved out-of-box performance. Review your saved templates to ensure they reflect your preferred configurations.

    Migration Guide¶

    Upgrading from Previous Versions

    Step 1: Back Up Your Current Setup

    • Screenshot your current indicator settings
    • Export any saved chart templates
    • Document your alert configurations

    Step 2: Remove Old Versions

    • Remove previous indicator versions from your TradingView charts
    • Clear any outdated alerts (they will not work with new versions)

    Step 3: Add New Versions

    • Add the v1.0 indicators from your TradingView invite-only indicators list
    • Look for the new Signal Pilot branding in indicator names

    Step 4: Reconfigure Settings

    • Review default settings (many have been optimized)
    • Adjust any custom parameters to match your previous setup
    • Note: Some setting names have changed for clarity

    Step 5: Update Alerts

    • Recreate alerts using the new alert conditions
    • Update alert message templates to use new signal names (BDN instead of OUT)
    • Test alerts on lower timeframes before deploying to live trading monitors

    Step 6: Verify Functionality

    • Compare signal output with your screenshots to ensure consistency
    • Monitor for 24-48 hours to confirm proper operation
    • Report any unexpected behavior to support@signalpilot.io

    Common Migration Issues

    • Issue: Alerts not firing
      Solution: Alerts need to be recreated with new v1.0 indicators (old alerts won't work)
    • Issue: Different signal output
      Solution: Review your settings against previous screenshots; default parameters may have changed
    • Issue: Cannot find indicator
      Solution: Check your TradingView invite status; contact support if access issues persist
    • Issue: Performance/loading issues
      Solution: Clear browser cache, refresh chart; v1.0 should be faster than previous versions
