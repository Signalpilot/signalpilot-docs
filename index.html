    🚀 Signal Pilot Suite (The Complete Cycle System)¶

    • Pentarch v1.0 — Five Cycle Events Identifies cycle position across TD (accumulation), IGN (markup), WRN (distribution), CAP (climax), BDN (decline). Four-layer detection system confirms before signals fire. Read Guide

    • Janus Atlas v1.0 — Levels System Displays HTF pivots, session boundaries, VWAP, volume profile nodes, and structural levels. Automatically updates at bar close. Read Guide

    • Omnideck v1.0 — Comprehensive System Combines TD Sequential, Squeeze, liquidity sweeps, EMAs, SuperTrend, BMSB, regime detection, zones, and pattern recognition. 10 systems with unified alerts. Read Guide

    • Augury Grid v1.0 — Multi-Symbol Screener Monitors 8 symbols simultaneously with quality scoring. On-chart table displays ranked opportunities at bar close. Read Guide

    • Volume Oracle v1.0 — Volume Intelligence Tracks institutional accumulation and distribution patterns through volume spike analysis. Includes position management framework with automatic entry, stop, and target levels. Read Guide

    • Harmonic Oscillator v1.0 — Composite Momentum Combines MACD, RSI, and StochRSI into unified 0-100 scale. Three-indicator voting system identifies momentum alignment and divergences. Read Guide

    • Plutus Flow v1.0 — Advanced OBV Spike-clipped On-Balance Volume with automatic divergence detection. Identifies institutional accumulation and distribution through volume-price relationships. Read Guide
