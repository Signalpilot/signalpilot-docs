    üîç Suite Overview ‚Äî The 7 Signal Pilot Indicators¶

    This page provides a comprehensive overview of all Signal Pilot indicators to help you understand what's available and choose the right tools for your trading style.

    The 7 Signal Pilot Indicators¶

    1. Pentarch v1.0 ‚Äî Reversal & Timing Signals¶

    What it does: Identifies potential cycle reversal events using TD Sequential methodology with 5 distinct signal types.

    Best for: Timing entries and exits, catching potential trend reversals, cycle-based trading

    Key Features: - 5 signal types (TD 9, TD 13, IGN, Perfection, Price Flip) - Non-repainting confirmations - Works on any timeframe and market

    Learn More ‚Üí

    2. Janus Atlas v1.0 ‚Äî Structure & Levels System¶

    What it does: Displays 39 different price levels including pivots, VWAP, session levels, and key zones with test count tracking.

    Best for: Level-based trading, support/resistance analysis, structure identification

    Key Features: - 39 customizable levels - Test count indicator (freshness tracking) - Cluster identification system

    Learn More ‚Üí

    3. Harmonic Oscillator v1.0 ‚Äî Momentum Voting System¶

    What it does: Combines 5 oscillators (RSI, Stochastic, CCI, Williams %R, MFI) with voting consensus for divergence detection.

    Best for: Momentum analysis, divergence trading, overbought/oversold confirmation

    Key Features: - 5-oscillator voting system - Automatic divergence detection - Consensus-based signals

    Learn More ‚Üí

    4. Plutus Flow v1.0 ‚Äî Advanced OBV (Volume Trend)¶

    What it does: Enhanced On-Balance Volume with automatic divergence detection (both regular and hidden).

    Best for: Volume trend analysis, institutional flow tracking, divergence confirmation

    Key Features: - Regular & hidden divergence signals - Smart OBV calculation - Volume-price relationship analysis

    Learn More ‚Üí

    5. Volume Oracle v1.0 ‚Äî Volume Analysis Suite¶

    What it does: Advanced volume analysis showing institutional accumulation/distribution with flow color indicators.

    Best for: Institutional flow tracking, volume confirmation, smart money analysis

    Key Features: - Flow direction indicators (green/red/neutral) - Accumulation/distribution zones - Volume spike detection

    Learn More ‚Üí

    6. Augury Grid v1.0 ‚Äî Multi-Symbol Scanner¶

    What it does: Scans up to 8 symbols simultaneously across multiple indicators with correlation heatmap.

    Best for: Finding trading opportunities across markets, portfolio diversification, multi-asset analysis

    Key Features: - 8-symbol scanning capacity - Correlation heatmap - Cross-market signal detection

    Learn More ‚Üí

    7. Omnideck v1.0 ‚Äî All-in-One Indicator (10 Systems)¶

    What it does: Combines 10 different trading systems in one indicator with customizable agreement thresholds.

    Best for: Comprehensive analysis, multi-system confirmation, all-in-one solution

    Key Features: - 10 integrated systems - Agreement voting system - Customizable system selection

    Learn More ‚Üí

    Choosing Your Indicator Setup¶

    üü¢ Beginner Setup (1-2 Indicators)¶

    Start with: Pentarch + Janus Atlas

    Why: Simple combination covering timing (Pentarch) and levels (Janus Atlas). Easy to learn and highly effective.

    Trading style: Swing trading, position trading

    üü° Intermediate Setup (2-3 Indicators)¶

    Recommended: Pentarch + Janus Atlas + Volume Oracle

    Why: Adds volume confirmation to your timing and level signals for higher probability setups.

    Trading style: Day trading, swing trading

    üî¥ Advanced Setup (3-4+ Indicators)¶

    Full stack: Pentarch + Janus Atlas + Harmonic Oscillator + Volume Oracle

    Why: Complete analysis covering cycles, levels, momentum, and volume for maximum confirmation.

    Trading style: Professional trading, multiple strategies

    üü£ All-in-One Alternative¶

    Use: Omnideck v1.0 alone

    Why: Get 10 systems in one indicator. Perfect if you want comprehensive analysis without multiple indicators.

    Trading style: Any style, one-indicator solution

    Example Trading Workflow (Pentarch + Janus + Volume)¶

    Here's how indicators work together in practice:

    1. Janus Atlas ‚Üí Identify key level (e.g., Daily pivot cluster at $100)
    2. Pentarch ‚Üí Wait for TD 9 signal at that level
    3. Volume Oracle ‚Üí Confirm with green flow (accumulation)
    4. Entry ‚Üí All 3 indicators align = high probability setup
